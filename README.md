## 📌 Project Overview
The **Supply Chain Transparency System** is a robust database solution designed to provide end-to-end visibility of products from production to delivery. By tracking every single batch of goods, the system ensures that quality standards are met, logistics are monitored in real-time using GPS, and compliance certifications are verified to maintain ultimate trust across the supply chain.

### 🎯 Key Requirements Solved:
- **Batch Tracking:** Records unique production batches and links them to suppliers and products.
- **Logistics Monitoring:** Real-time tracking of shipments using GPS coordinates and timestamps.
- **Quality Assurance:** Mandatory inspection processes with Pass/Fail results and defect reporting.
- **Compliance Management:** Tracks ISO, Organic, and other certifications for suppliers and batches.
- **Personnel Management:** Role-based staff assignments (Drivers, Inspectors, Warehouse Managers).

## 💻 How to Run This Project
1. Clone this repository to your local machine.
2. Open your preferred SQL database management tool.
3. Import and execute the `.sql` script file provided in this repository to automatically generate the database schema, tables, relationships, and mock data.
4. Run the analytical queries provided in the queries folder to test the system's tracking and verification functionalities.
